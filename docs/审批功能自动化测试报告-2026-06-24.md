## 🔧 测试过程中发现的技术问题

### 问题1: Element Plus 下拉框遮挡问题

**现象**: 在用户创建对话框中，选择角色后下拉框不会自动关闭，导致遮挡"保存"按钮无法点击

**原因**: Element Plus 的 Select 组件在某些情况下不会自动关闭下拉菜单

**解决方案**: 
```javascript
// 选择选项后主动按 ESC 键关闭下拉框
await page.click('.el-select-dropdown:visible .el-select-dropdown__item:first-child');
await page.waitForTimeout(500);
await page.keyboard.press('Escape');  // 关键
await page.waitForTimeout(500);
```

**影响**: 仅影响自动化测试，手动操作时用户会自然点击其他区域关闭下拉框，无实际用户体验问题

---

### 问题2: 登录表单选择器差异

**现象**: 初始测试脚本使用 `input[type="text"]` 无法找到用户名输入框

**原因**: 登录表单使用的是 `input[name="account"]` 而非通用的 `type="text"`

**解决方案**: 通过调试脚本遍历多种选择器，最终确定正确的选择器
- 用户名: `input[name="account"]`
- 密码: `input[type="password"]`
- 登录按钮: `button:has-text("登录")`

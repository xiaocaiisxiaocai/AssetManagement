- [ ] **Step 4: 实现 CategoryCodeService（纯函数）**

```csharp
namespace AssetManagement.Domain.Services;
public static class CategoryCodeService
{
    public static string Compose(string? parentCode, string seg)
        => string.IsNullOrEmpty(parentCode) ? seg : $"{parentCode}-{seg}";

    // 递归重算 node 及其子孙 Code（node 形态由调用方提供 Seg/Code/Children 访问器）
    public static void Recalc(ICatNode node, string? parentCode)
    {
        node.Code = Compose(parentCode, node.Seg);
        foreach (var c in node.Children) Recalc(c, node.Code);
    }
}
public interface ICatNode { string Seg {get;set;} string Code {get;set;} IEnumerable<ICatNode> Children {get;} }
```
（测试辅助 `CatNode` 实现 `ICatNode`。）

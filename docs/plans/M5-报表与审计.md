# M5 报表与审计 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development 或 superpowers:executing-plans。**前置：M3、M4 完成。**

**Goal:** 实现资产汇总（按类别/部门）、借用明细、逾期资产三类统计报表与导出，审计日志查询与导出，逾期资产站内手动催办（无邮件、无定时任务）。

**Architecture:** 报表为只读聚合查询（EF GroupBy / 原生聚合），导出复用 M3 的 EPPlus。审计日志读 M1 写入的 `audit_logs`。催办写站内通知记录（简单表或复用审计），不发邮件。

**Tech Stack:** EF Core 聚合查询、EPPlus。

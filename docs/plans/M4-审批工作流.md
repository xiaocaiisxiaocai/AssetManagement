# M4 审批工作流引擎 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development 或 superpowers:executing-plans。**前置：M2、M3 完成。** 本里程碑最复杂，核心是工作流引擎领域逻辑，务必先 TDD。

**Goal:** 实现可配置审批工作流：流程定义（流程图式设计器）、运行时实例按节点流转（会签/或签/条件分支/抄送）、审批操作（通过/驳回/加签/转签），完成后执行业务动作（借用→借出、转让→归属转移、归还→入库）。

**Architecture:** `Workflow`（流程定义，节点存 JSON 列）；发起时克隆节点为 `ApprovalFlow` 实例（节点快照存 JSON + `CurrentNodeIndex`）。`WorkflowEngine` 领域服务承载推进/判定逻辑（纯逻辑，TDD）。业务动作通过 `IBizEffectApplier` 解耦。设计参见 `docs/审批工作流设计.md`，行为与原型 `app.js` 一致。

**Tech Stack:** EF Core（JSON 列 `HasConversion`/`ToJson`）、System.Text.Json。

    // 单人审批通过 → 推进到下一节点

        WorkflowEngine.Approve(f, "张三", "ok");

    }
    // 条件：amount 不满足→跳过
    [Fact] public void Condition_skipped_when_not_met() {
        var f = Flow(N("发起",Done), N("主管",Current,"approval"), Cond("副总",Pending,"amount>5000"), N("结束",Pending,"end"));

        WorkflowEngine.IsFinished(f).Should().BeTrue();
    }
    // 条件满足→经过
    [Fact] public void Condition_kept_when_met() {
        var f = Flow(N("发起",Done), N("主管",Current,"approval"), Cond("副总",Pending,"amount>5000"), N("结束",Pending,"end"));

- [ ] **Step 3: 实现 WorkflowEngine**

```csharp
namespace AssetManagement.Domain.Workflow;
public static class WorkflowEngine
{
    public static FlowInstanceNode Current(ApprovalFlow f) => f.Nodes[f.CurrentNodeIndex];
    public static bool IsFinished(ApprovalFlow f) => f.Status == "approved";

    public static void Approve(ApprovalFlow f, string? signer, string opinion)
    {
        var node = Current(f);
        if (node.Type is NodeType.Countersign or NodeType.Orsign)
        {
            node.SignStates ??= new();
            var who = signer ?? node.Approver ?? "";
            node.SignStates[who] = true;
            var all = (node.Signers ?? new()).Concat(node.AddedSigners ?? new()).ToList();
            var remaining = all.Where(s => !node.SignStates.GetValueOrDefault(s)).ToList();
            if (node.Type == NodeType.Countersign && remaining.Count > 0) return; // 停留
        }
        node.Status = NodeStatus.Done; node.Opinion = opinion; node.Time = "刚刚";
        Advance(f);
    }

    private static void Advance(ApprovalFlow f)
    {
        for (int i = f.CurrentNodeIndex + 1; i < f.Nodes.Count; i++)
        {
            var n = f.Nodes[i];
            if (n.Type == NodeType.End) { n.Status = NodeStatus.Done; f.CurrentNodeIndex = i; f.Status = "approved"; return; }
            if (n.Type == NodeType.Condition && !EvalCondition(n.Condition, f)) { n.Status = NodeStatus.Skipped; continue; }
            if (n.Type == NodeType.Cc) { n.Status = NodeStatus.Done; n.Opinion = "已知会"; continue; }
            n.Status = NodeStatus.Current; f.CurrentNodeIndex = i; return;
        }
        f.Status = "approved";
    }

    public static bool EvalCondition(string? cond, ApprovalFlow f)
    {
        if (string.IsNullOrEmpty(cond)) return true;
        var m = System.Text.RegularExpressions.Regex.Match(cond, @"amount\s*>\s*(\d+)");
        return m.Success ? f.Amount > decimal.Parse(m.Groups[1].Value) : true;
    }

    public static void Reject(ApprovalFlow f, string reason)
    { Current(f).Status = NodeStatus.Rejected; f.Status = "rejected"; }

    public static void AddSign(ApprovalFlow f, string who)
    {
        var node = Current(f);
        (node.AddedSigners ??= new()).Add(who);
        if (node.Type is NodeType.Countersign or NodeType.Orsign) (node.Signers ??= new()).Add(who);
    }

    public static void Transfer(ApprovalFlow f, string who) => Current(f).Approver = who;
}
```

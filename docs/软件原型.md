## 4. 资产列表

### 4.1 视图模式切换

资产列表提供两种查看模式：
- **📊 列表视图**：传统表格形式，适合批量操作和导出
- **🗂️ 层级视图**：按编号层级卡片式导航，适合按编号查找资产

用户可通过页面右上角切换按钮在两种视图间切换。

### 8.2 审批操作说明

**通过操作**：
- 审批意见为选填
- 点击"确认提交"后：
  - 工单状态变更为"已通过"
  - 资产状态变更为"借出中"
  - 保管人变更为申请人
  - 系统发送邮件通知申请人

**驳回操作**：
- 驳回理由为必填（10-200字）
- 点击"确认提交"后：
  - 工单状态变更为"已驳回"
  - 资产状态保持不变
  - 系统发送邮件通知申请人

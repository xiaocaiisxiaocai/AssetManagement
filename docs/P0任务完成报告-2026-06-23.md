### ✅ P0.2 - 前端请求防抖/防重复提交

**状态**: 已完成  
**耗时**: 约 1 小时

#### 完成内容

已为以下关键操作添加 300ms 防抖:

1. **资产管理 (AssetFormDialog.vue)**
   - 创建/编辑资产保存按钮
   - 使用 `useDebounceFn(save, 300)`

2. **资产列表 (list/index.vue)**
   - 删除资产按钮
   - 使用 `useDebounceFn(remove, 300)`

3. **待审批 (approval/pending/index.vue)**
   - 审批通过按钮
   - 审批驳回按钮
   - 使用 `useDebounceFn(approve, 300)` 和 `useDebounceFn(reject, 300)`

4. **确认入库 (approval/confirm-return/index.vue)**
   - 确认入库按钮
   - 使用 `useDebounceFn(confirmReturn, 300)`

#### 技术方案
```typescript
import { useDebounceFn } from '@vueuse/core';

// 原函数
async function save() { ... }

// 防抖版本
const debouncedSave = useDebounceFn(save, 300);

// 模板中使用
<ElButton @click="debouncedSave">保存</ElButton>
```

#### 效果
- 防止用户快速点击导致重复提交
- 300ms 防抖窗口,体验流畅
- 覆盖所有关键写操作

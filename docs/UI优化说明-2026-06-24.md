## 🔧 使用方法

### 标准页面结构

```vue
<template>
  <re-page>
    <div class="page-container">
      <!-- 页面头部 -->
      <div class="page-header">
        <div>
          <h2 class="page-title">页面标题</h2>
          <p class="page-subtitle">页面描述</p>
        </div>
        <ElButton type="primary">操作按钮</ElButton>
      </div>

      <!-- 筛选面板（可选） -->
      <div class="filter-panel">
        <ElForm class="filter-form" inline>
          <!-- 筛选表单 -->
        </ElForm>
      </div>

      <!-- 表格面板 -->
      <div class="table-panel">
        <ElTable :data="list" border>
          <!-- 表格列 -->
        </ElTable>
      </div>
    </div>
  </re-page>
</template>

<style scoped>
/* 仅需编写页面特有的样式，统一样式已自动应用 */
</style>
```

### 带工具栏的表格

```vue
<div class="table-panel-with-toolbar">
  <div class="table-toolbar">
    <span class="table-total">共 {{ total }} 条</span>
    <ElSelect v-model="pageSize">
      <!-- 分页选项 -->
    </ElSelect>
  </div>
  
  <ElTable :data="list" border>
    <!-- 表格列 -->
  </ElTable>
  
  <div class="table-pagination">
    <ElPagination />
  </div>
</div>
```

### 统计卡片

```vue
<div class="stat-cards">

    <div class="stat-label">资产总数</div>
    <div class="stat-value">{{ total }}</div>

  <!-- 更多卡片 -->
</div>
```

## 🚀 优化步骤（供后续参考）

### 优化现有页面

1. **替换容器类名**:
   ```vue
   <!-- 旧 -->
   <div class="custom-page">
   
   <!-- 新 -->
   <div class="page-container">
   ```

2. **替换头部类名**:
   ```vue
   <!-- 旧 -->
   <div class="custom-header">
     <h2 class="custom-title">标题</h2>
   
   <!-- 新 -->
   <div class="page-header">
     <div>
       <h2 class="page-title">标题</h2>
       <p class="page-subtitle">副标题</p>
     </div>
   ```

3. **替换表格面板类名**:
   ```vue
   <!-- 旧 -->
   <div class="custom-table-panel">
   
   <!-- 新 -->
   <div class="table-panel">
   ```

4. **删除重复的样式代码**:
   - 删除 `<style scoped>` 中与 `design-system.css` 重复的样式
   - 仅保留页面特有的样式（如特殊布局、业务逻辑相关样式）

## ✨ 优势

1. **统一性**: 所有页面使用相同的设计规范，视觉体验一致
2. **可维护性**: 样式集中管理，修改设计规范只需更新一个文件
3. **开发效率**: 新页面直接使用现成的样式类，无需重复编写
4. **代码简洁**: 单个页面样式代码大幅减少，可读性提升
5. **响应式**: 内置响应式布局支持，自动适配不同屏幕

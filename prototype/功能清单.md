## 🚀 使用说明

1. **打开原型**：双击 `index.html` 文件
2. **登录**：输入任意工号和密码
3. **导航**：点击左侧菜单切换页面
4. **操作**：所有按钮、表单、筛选、分页都可点击操作

3. **完整交互** - 所有按钮、表单都有实际功能
4. **表单验证** - 前端验证+错误提示
5. **视觉反馈** - 操作后有明确提示
6. **数据可视化** - 进度条、饼图、统计卡片
